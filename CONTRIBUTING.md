# Contributing to (NexusLLM)
Welcome to the NexusLLM project, this is a open source community effort to build a custom transformer-based language model (MinGPT). I'm so happy you are here to help! This will rule out everything on what you have to do in order to meet our expectations. 

## Rules:
1. **Respect The Project:**
NexusLLM is built from scratch, so please keep things simple and understandable so that we all have clarity.

2. **Don't Overengineer:**
This is *not* a big project, so please keep the code minimal and readable.

3. **Use Plain text:**
The training data used for the AI should be kept in '.txt' (UTF-8). If you are helping to make more information in the training data, please keep it copyright free and offensive free.

4. **Be Respectful:**
Do not troll, spam, or be toxic. Otherwise, you will not be participating in this project. Simple as that.

## How to Contribute:
1. **Fork The Repository:**
If you want to help, change, or improve anything in the code, make sure to fork the repository and I'll have a look at it. And make sure to write clear informative commitments on what you did to the code.
   
2. **Submit Requests:**
If you want to request a feature, make sure to do through GitHub's Issue tab. It could to imrprove something, add traning tools, etc.

3. **Code Of Conduct:**
Make sure to be respectful, even if your disagreeing on something you don't like you need to be calm about it. Always make sure to help others learn, this is how we all connect with each other even if they don't understand it.
