# Contributing to (NexusLLM)
Welcome to the NexusLLM project, this is a open source community built toy transformer. I'm so happy you are here to help!
This will rule out everything on what you have to do in order to meet our expectations. 

## Rules:
1. **Respect The Project.**
NexusLLM is a small trasformer, built from stratch. Please keep things simple and understandable so tha we all have clarity.

2. **Don't Overengineer.**
This is *not* a big project, so please keep the code as minimal and readable.

3. **Stick To NumPy For Now.**
This is the core of the project so no PyTorch, TensorFlow or anyother frame until we say so.

4. **Use Plain text**
The training data used for the AI should be kept in '.txt' (UTF-8). If you are helping to make more information in the training data, please keep it copyright free and offensive free.

## How to Contribute:
*If you want to help or change anything in the code, make sure to fork the repository and I'll have a look at it.*
